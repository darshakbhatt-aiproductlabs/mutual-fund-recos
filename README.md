# mutual-fund-recos

Sector & style (factor) return history for Indian equity markets, a
strategy picker (contrarian / momentum / balanced), and data-driven
mutual fund recommendations matched to the chosen strategy.

**Status:** scaffold only — Vite + React app boots, no data or logic yet.

See [`HANDOFF.md`](./HANDOFF.md) for the full build spec, architecture,
and decisions already locked in. That file is written as a ready-to-paste
brief for an agentic coding session (e.g. Claude Code).

## Local dev
```bash
npm install
npm run dev
```
